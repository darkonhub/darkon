Installation
------------

Install Darkon alone

.. code-block:: bash

   pip install darkon

Install with TensorFlow CPU

.. code-block:: bash

   pip install darkon[tensorflow]

Install with TensorFlow GPU

.. code-block:: bash

   pip install darkon[tensorflow-gpu]
